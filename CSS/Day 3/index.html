Inline CSS has the highest priority.

"id" has the second highest priority.

"class" has the third highest priority.

"tags" (h1, h2, div etc..) has the fourth highest priority.

If the link tag appears before the internal CSS then the priority will be given to internal CSS because it appears after the link tag in which external CSS is linked. Reverse is also possible.
